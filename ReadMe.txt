# Twitter Sentiment Analysis using Tweepy
Prerequisities - 
1. Install tweepy using pip
2. Python v3.6 (not v3.7) to be able to run Tweepy library
3. An active Twitter account with CONSUMER_API_KEY, CONSUMER_API_SECRET, ACCESS_TOKEN, ACCESS_TOKEN_SECRET

The code will connect to Twitter using the above mentioned details and get a few tweets. Further it will analyse the sentiment of the tweet using library tweepy and print out the output as a negative/ positive/ neutral sentiment with respect to the topic provided as input.

Note: <Your Value From Twitter account> indicates that one needs to replace these with the values that are available on their respective twitter accounts.
